# Logo Image Instructions

## Where to Place Your Logo

Place your TOPS TECHNOLOGIES logo image file in this directory (`frontend/public/images/`) with the filename:

**`tops-logo.png`**

## Supported Formats

- PNG (recommended for transparency)
- JPG/JPEG
- SVG (scalable vector graphics)

## Logo Specifications

The logo should be:
- High resolution (at least 400px width recommended)
- Transparent background (PNG with alpha channel) preferred
- Clear and readable at different sizes

## Current Logo Design

Based on the TOPS TECHNOLOGIES logo description:
- Light blue 3D cube icon with arrows
- "TOPS" text in light blue, uppercase, bold
- "TECHNOLOGIES" text in black, uppercase, bold
- Tagline: "BUILDING CAREERS SINCE 15+ YEARS" in white on black bar

## Where the Logo Appears

1. **Login/Registration Page** - Displayed prominently at the top
2. **Navbar** - Shown in the navigation bar on all authenticated pages

## Fallback

If the logo image is not found, a text-based fallback will be displayed automatically.
